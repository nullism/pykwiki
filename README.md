PyKwiki
=======

PyKwiki is a static content generation engine for Python. 

Features:

    * Static keyword search
    * YAML configuration
    * Markdown
    * Templates
    * Extensions
    * SASS (version 2+)
    

## Install

To install Pykwiki 1x:

    pip install -I pykwiki==1.1.6

To install the latest version: 

    pip install pykwiki
